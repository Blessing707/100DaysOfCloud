# Day 1 — Create Key Pair

**Date:** 2026-03-15
**Resource:** KodeKloud Engineer

## What I Did
Created a Key Pair in AWS. A Key Pair is used to securely 
connect to EC2 instances via SSH.

## What I Learned
- A Key Pair consists of a public key (stored by AWS) and a 
  private key (downloaded to your computer)
- Without the Key Pair you cannot SSH into your EC2 instance
- The private key file (.ppk) must be kept safe — AWS will 
  never give it to you again
